About FSD

Full Stack Development (FSD) is a comprehensive program that equips individuals with the skills and knowledge needed to build and manage both the Frontend and back-end of web applications. This course covers a wide range of technologies, including HTML, CSS, JavaScript, databases, server management, and more. Students will learn how to create dynamic, responsive websites and applications, as well as how to deploy and maintain them effectively.

Technologies covered

  • HTML5
  • CSS3
  • JavaScript
